Создание тестовой базы
----------------------

psql -d pulxer_test -f CreateDb.sql - исполнение скрипта



Сгенерировать скрипт для создания пустой базы
---------------------------------------------

pg_dump -s pulxer > createdb.sql - запись скрипта создания в файл (только схема, без данных)

pg_dump -s -t <table> pulxer > <table>.sql - скрипт создания одной таблицы
